# Support

This document explains where to go for different types of support.

## Product support

For customer-facing support, please use the relevant product’s site:

- Pacific North North: https://pacificnorthnorth.com
- Fundr.Science: https://fundr.science
- DungeonParty.ai: https://dungeonparty.ai

## Developer support

If you’re working with Yeix.io open-source repos:

- Check the project’s `README.md` and `/docs` folder.
- Search existing issues to see if your question has already been answered.
- If not, open a **Discussion** or an issue in the relevant repo with:
  - What you’re trying to do
  - What you expected
  - What actually happened (logs, stack traces, etc.)

For security-related questions, see [`SECURITY.md`](./SECURITY.md).
